Goal

The intent of this document is to provide the reader with the available versions of the Sun SPARC(R) Enterprise M3000, M4000, M5000, M8000, and M9000 XCP Firmware Image software (XCP) and to inform where and how to obtain the appropriate images.

To cross reference bugs to the XCP versions containing the fix, please refer to "SPARC Enterprise M3000/M4000/M5000/M8000/M9000 - XCP Firmware to Defect Cross Reference".

To view the table with supported hardware, firmware and operating system (OS) combinations, as well as required software patches for M-series servers, please read "SPARC Enterprise M3000/M4000/M5000/M8000/M9000 Server Matrix for Hardware, Firmware and Software (Doc ID 1145383.1)" .

XCP firmware consists of three components, the XSCF firmware, the OpenBoot PROM (OBP) firmware and the Power On Self Test (POST) firmware. After downloading the appropriate patch for your system, download the XCP image onto your XSCF using either the Browser User Interface (BUI) or the getflashimage command. After succesfully downloading the XCP image onto the XSCF, run the flashupdate command to install the new XCP firmware onto the XSCF. Finally, in case the update contains a new version of POST or OBP, each domain needs to be rebooted to activate the new POST/OBP. For more details, please refer  to the Browser Users Interface (BUI) ,  getflashimage and flashupdate  documentation.

Please refer to the available XSCF user guide and release notes for: M8000 / M9000 Product Library Index , M4000 / M5000 Product Library Index and M3000 Product Library Index

 

To discuss this information further with Oracle experts and industry peers, we encourage you to review, join or start a discussion in the My Oracle Support Community - M Series Servers

Solution

 

Steps to Follow
The XSCF Control Package firmware image (or XCP for short) is the software image that is downloaded and installed on to a Sun SPARC Enterprise M3000 through M9000 Extended System Control Facility Unit (XSCFU) enabling the XSCFU to function as a Service Processor.

Once downloaded the patch will contain the XCP image named:

     The M3000 XCP image (i.e.  IKXCP.tar.gz )

     The M4000/M5000 XCP image (i.e.  FFXCP.tar.gz )

     The M8000/M9000 XCP image (i.e.  DCXCP.tar.gz )

It is very important that the M3000 XCP image or M4000/M5000 XCP image are not installed on a Sun SPARC  Enterprise M8000/M9000 XSCF and vice versa.

SNMP MIB and other optional files will now be included within the patch package.  It is important to read and understand the SNMP documentation and added functionality of this package prior to installing and enabling the SNMP MIB package.  

NOTE: XCP 1093 is the minimum requirement for a domain running Solaris 10 9/10 with the processor groups observability feature enabled. Please refer to Solaris 10 9/10 Product Notes for more details about this feature.

Where to locate patches, updates, and/or firmware images?

Product patches, updates and firmware are available for download from My Oracle Support, on the Patches and Updates tab across the top of the page (below where your user name or email is displayed once logged in).
SEARCH TIPS:  Search for the patch using the Advanced Search option and choose to filter the results based on your system type:
For example choose Sun SPARC Enterprise M3000 or Sun SPARC Enterprise M8000 or whatever system you are looking for an image for.
After selecting the appropriate system, the next drop-down menu will ask you to sort through a list of XCP releases and select the desired release (we suggest the latest). 

After selecting the XCP version and clicking search,  a listing will appear that shows you the patch ID which you can open and download the firmware from.
Information on accessing and using My Oracle Support can be found at the My Oracle Support Welcome Center for Oracle Sun Customers and Partners.
